EcoCar Zim is a progressive web app designed to support the adoption of electric vehicles in Zimbabwe.

## Features

- Interactive map of EV charging stations
- Business directory for EV-friendly establishments
- Route planner for electric vehicles
- Community reporting system for station status

## Tech Stack

Built with Astro for the frontend and a lightweight Node.js API for real-time station data.

## Impact

The app has helped reduce "range anxiety" for EV owners in Zimbabwe and has been downloaded over 2,000 times since launch.
